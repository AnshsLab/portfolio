Neural Network Visualizer

Abandoned • 2023
What I Was Trying to Do

Build an interactive 3D visualization tool for understanding how neural networks process data. The goal was to animate the flow of information through layers and highlight activated neurons in real-time.

What I Learned

Gained deep understanding of Three.js for 3D rendering, WebGL performance optimization, and the mathematics behind backpropagation. Learned that visualizing complex ML concepts requires both technical skill and thoughtful UX design.

Why It Stopped

The computational complexity of real-time visualization exceeded browser capabilities for larger networks. Would need to pivot to a different architecture or simplify the scope significantly.

Technologies
Three.js WebGL TensorFlow.js React

AI Portrait Generator

Paused • 2024
What I Was Trying to Do

Create a fine-tuned diffusion model pipeline that could generate stylized portraits on-demand from a simple text prompt, optimized for web delivery with a clean UI on top.

What I Learned

Deep knowledge of diffusion model fine-tuning, LoRA adapters, and inference optimization. Learned a lot about GPU memory constraints and batching strategies for consumer hardware.

Why It Stopped

Inference costs at scale were prohibitive. The gap between local results and production-quality output required infrastructure investment that wasn't justified for a side project.

Technologies
Python Diffusers LoRA FastAPI

Real-Time Analytics Dashboard

On Hold • 2024
What I Was Trying to Do

Build a comprehensive analytics dashboard with real-time data visualization, customizable widgets, and advanced filtering — intended for monitoring business metrics across multiple data sources.

What I Learned

Deep dive into D3.js and data visualization best practices. Efficient state management for large datasets, websocket optimization, and creating performant reusable chart components.

Why It Stopped

Scope creep became overwhelming. The feature list grew too large and real-time data synchronization across multiple sources proved far more complex than initially anticipated.
